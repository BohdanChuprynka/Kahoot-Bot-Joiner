# Kahoot Flooder
A bot that floods a specified amount of bots to a Kahoot game. It was developed for educational purposes only.

### Installation: 

1. Clone the repository, open it in your working terminal, and ensure the path is linked to current working repository.
```
git clone https://github.com/BohdanChuprynka/Kahoot-Flooder.git
```

3. Install dependencies: 
```
npm install
```

2. Run the script:

On the server:
```
node botServer.js
```

Or in the terminal: 
```
node terminal/index.js
```
