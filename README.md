# Kahoot-Bot-Joiner
A bot that floods a specified amount of bots to a Kahoot game. It was developed just for educational purposes only.

### Installation: 

1. Clone the repository, open your working terminal, and link it where the new repository located. Then, follow next step.

2. Install dependencies: 
```
npm install
```

2. Run the script:

On the server:
```
node botServer.js
```

Or in the terminal: 
```
node terminal/index.js
```
